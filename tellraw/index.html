How to Use

Tellraw Commands:

  1. Configure your message text, colors, and formatting
  2. Add click events and hover text if desired
  3. Generate and copy the command
  4. Paste into command blocks or chat

Title Commands:

  1. Choose title type (main, subtitle, or action bar)
  2. Set text, colors, and timing
  3. Add optional sound effects
  4. Generate and use the command

Function Files:

  1. Add multiple commands to build a function
  2. Download as .mcfunction file
  3. Place in datapacks/[name]/data/[namespace]/functions/
  4. Use with /function command
